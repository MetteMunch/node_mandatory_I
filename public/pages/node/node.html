Node.js

Node.js er et runtime miljø for JavaScript. Fordelen er, at vi som udviklere kan bruge det samme sprog (JavaScript) fullstack, dvs. både på client-side og server-side.


Du kan finde og downloade seneste version af Node.js her:

officiel Node.js webpage

Du kan tjekke hvilken version af Node.js du har installeret med:

node -v
