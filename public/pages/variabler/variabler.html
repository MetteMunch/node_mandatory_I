Variabler

En variabel er en form for beholder til noget data af en given type.

Datatyperne i JavaScript er:

  • Number
  • String
  • Boolean
  • Object
  • BigInt
  • Symbol
  • undefined
  • null

Når en variabel navngives (deklareres), så benyttes enten const eller let (undgå var) og navngivning er med camel case: 

let myAge;

Variablen tildeles en værdi (initialiseres) med et lighedstegn:

myAge = 51;

Oftest vil deklarationen og initialiseringen ske samtidig:

let myAge = 51;

En const variabel skal naturligvis initialiseres med det samme:

const myName = "Mette";

Om variablen skal være const eller let er afhængig af om variablen skal kunne tildeles en ny værdi indenfor samme scope.

En let variabel kan få ændret værdien:

let letVar = 1;
function someFunction() {
    letVar ++;    
    console.log(letVar); // output: 2
}

const constVar = 1;
function someFunction() {
    constVar ++;
    console.log(constVar);   // TypeError: Assignment to constant variable 
}

Men selve indholdet altså elementerne i objekter og arrays kan ændres selvom det er gemt i const variabel: 

const array = [1,2,3,4];
function someFunctionNew() {
    array.push(5);
    console.log(array); //output: [1,2,3,4,5]
}

Men en ny initialisering er ikke mulig:

const newArray = [1,2,3,4];
function newFunction() {
    newArray = [5,6,7];
    console.log(newArray); // TypeError: Assignment to constant variable
}

JavaScript er et dynamisk kodesprog, hvor datatypen ikke klart angives på variablen, men afhænger af værdien. Hvilket betyder, at den samme variabel reelt kan bruges til at gemme flere forskellige datatyper. I tvivl om datatypen, så brug eventuelt typeof. 

let varType = "String";
console.log(typeof varType); // output: string
varType = 256;
console.log(typeof varType); // output: number
varType = true;
console.log(typeof varType); // output: boolean
varType = { name: "Mette"};
console.log(typeof varType); // output: object
