Server

Når vi ønsker at håndtere HTTP anmodninger (requests og responses), så skal vi bruge en server til dette.

Express framework

Express er et populært webframework til Node.js. Når vi opsætter serveren, så gøres dette i en app.js fil, hvor vi i korte træk importerer biblioteket / Express framework, instantierer/opretter et Express-applikationsobjekt (app) og angiver hvilken port, serveren skal lytte efter HTTP kald på.

Node ES Module

Den gængse metode nu er at anvende ES Module, men der vil stadig være meget kode, som anvender commonJS (se nedenstående kode eksempel med require). Husk for at anvende ES Module, så skal package.json indeholde: "type": "module". 


      import express from 'express';

      const app = express();

      app.use(express.static("public")); 

      app.get('/', (req, res) => {
      res.send('Hello World');
      });

      const PORT = 8080;
      app.listen(PORT, () => console.log('Server kører på port:', PORT));

CommonJS (ældre standard)


      const express = require('express');
      const app = express();

      app.get('/', (req, res) => {
      res.send('Hello World');
      });

      const PORT = 8080;
      app.listen(PORT, () => console.log('Server kører på port:', PORT));
