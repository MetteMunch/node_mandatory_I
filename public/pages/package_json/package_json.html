Package.json

Package.json er ligesom en opskrift for projektets afhængigheder og kan også indeholde andre metadata plus eventuelle scripts.


package.json kan oprettes manuelt ude i roden af projektet (der hvor også serveren / app.js ligger), og fx bare indeholde: 

{
      "dependencies": {
      "express": "^5.1.0"
      },
      "type": "module"
      }

Eller man kan oprette den ved kommandoen:

npm init

Denne kommando opretter automatisk en package.json fil, hvor der skal svares på en masse spørgsmål i forhold til metadata.

Efter oprettelse så kør:

npm install

Så indlæses fra NPM registry de moduler og undermoduler som er nødvendige i forhold til de angivne afhængigheder / dependencies i package.json filen. Disse pakker placeres i mappen node_modules/ og samtidig oprettes package-lock.json filen. Hvis der efterfølgende skal tilføjes en ny afhægighed, så kan dette indskrives direkte i package.json filen, hvorefter der igen køres en npm install. Eller ved direkte at køre npm install "dependency_name".
