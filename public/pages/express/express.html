Express

Express er et framework til Node.js, der gør det meget nemmere at lave en webserver. Med dette framework kan vi fx nemt håndtere HTTP-anmodninger (GET, POST, osv.), middleware og routing.


Installering

Når et Node.js projekt er oprettet (package.json fil), så kan Express installeres med nedenstående kommando, eller det kan skrives direkte ind i package.json filen under dependencies.

npm install express
