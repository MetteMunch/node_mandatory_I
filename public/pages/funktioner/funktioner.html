Funktioner

Almindelig funktion:

function rollDice(min, max) {
    return Math.floor(Math.random() * (max - min + 1) + min);
}

Anonym funktion:

const rollAnotherDice = function (min, max) {
    return Math.floor(Math.random() * (max - min + 1) + min);
}

Arrow function:

const rollYetAnotherDice = (min, max) => {
    return Math.floor(Math.random() * (max - min + 1) + min);
}

Callback function:

En callback funktion er en funktion, der videregives som et argument til en anden funktion og er beregnet til måske at blive eksekveret senere. I nedenstående er congrats() en callback funktion.
