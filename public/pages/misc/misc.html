Hvad er hvad?

Lidt om begreberne Routes, Pages, API og middleware

Routes

Ruter / stier der håndteres af Express (fx ”/” eller den dynamiske ”/:page” som begge anvendes i denne applikation).

Pages

Er routes som returnerer HTML-sider.

API-endpoints

Er routes som returnerer data (JSON).

Middleware

Kan tænkes som hjælpe-funktioner, der kører mellem request fra klienten og response fra serveren, og disse er tilgængelige via Express frameworket. Et eksempel på en Middleware funktion er: 

app.use(express.static("public"));

Denne sikrer, at statiske filer fra den angivne mappe (her ”public”), bliver tilgængelige direkte, så der ikke skal angives en route for hver af disse.
